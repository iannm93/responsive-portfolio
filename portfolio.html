Are You Still Watching?

Are You Still Watching is a web program that shows users where their favorite television shows and movies can be streamed, and gives the user a direct link to their streaming service. This application utilizes 3 different APIs; Utelly, OMDB, and YouTube to give users information such as: the plot of the movie/show, when the movie/show was released, the movie/show's poster, the IMDB ratings, and returns an embeddded trailer from YouTube. My reason for creating this application was to help users find the movies and shows they're interested in watching without having to go through the multiple steps that sometimes entails. Are You Still Watching will lead you right to the action!

Work Day Scheduler

This is a web application I designed to help a user keep track of their busy work day. This application uses local storage to store the user's input in the time slots so that when they return to the page, all of their important shceduled information will still be available to them. This application also helps users keep track of the time by dynamically updating the color of the time slots based on the current time of day. When the time of day has passed, the times turn grey, when the time is yet to come, the color is green, and when it's on the current hour, that time slot is red. An organized day is a more productive day!

Random Password Generator

The Random Password Generator is a web application that prompts the user with a series of questions that it uses to return a completely randomized password when the user clicks "generate password". The user can choose between numbers, capital, lower-case, specials, numerics, and a length between 8 and 128 characters. This application is useful, especially for anyone who in tech or who uses multiple online accounts, as having secure, unique, randomized passwords is vital to keeping your personal information secure.

Code Quiz

The Code Quiz is a web application I designed to test a user's basic understanding of HTML/JS/CSS through a small quiz. It's a multiple choice quiz, and every time a user gets a question right, their score goes up by one. When the user reaches the end of the quiz, they can enter their score, and then they're redirected to a "high score" page where their answers will be saved using local storage. It's a great and fun way to quickly check your understanding of basic concepts!
